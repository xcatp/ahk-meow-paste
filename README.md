# 介绍

一个使用 Ahk2 编写的贴图小工具，简洁且优雅。

运行`MeowPaste.ahk`以开始。

# 热键

- !\`   开始截图（!指Alt）
- ESC  取消截图
- !ESC 清除贴图
- \`    切换光标捕获（仅开始阶段）

# 功能

下面列举了当前包含的功能。

## 基本功能

1. 截图
2. 贴图
3. 保存为文件
4. 读取文件
5. 文件分组

## 辅助功能

1. 完全自定义：所有重要参数都可配置
2. 快速贴图：将图片文件拖放到贴图ui上可快速贴图
3. 贴图操作：包括**缩放、翻转**
4. 添加时间戳：根据在贴图中的点击位置生成时间戳
5. 国际化：也可以自己修改文句文件
